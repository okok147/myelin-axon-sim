Myelin / White-Matter Conduction Simulator

Educational/research simulator for saltatory conduction with modular disease patterns (MS, NMOSD, MOGAD, ADEM, PML, ALD, KRABBE, MLD, GBS, CIDP, CMT).

Repository: github.com/okok147/myelin-axon-sim

Run locally: python -m src.simulate --preset MS --sync --output_dir outputs
